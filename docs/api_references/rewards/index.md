# Rewards API Reference

## Overview

AgentFly provides a flexible reward system for training agents with reinforcement learning.

## Structure

- [Reward](reward.md) - Base reward class and decorator
- [Code Reward](code_reward.md) - Code execution rewards
- [Math Reward](math_reward.md) - Mathematical reasoning rewards
- [QA Reward](qa_reward.md) - Question answering rewards
- [AlfWorld Reward](alfworld_reward.md) - AlfWorld environment rewards
